# astrbot_plugin_kfc_crazy_thursday

> 疯狂星期四秦始皇马甲回复插件

当 UMO 白名单内的会话中出现"疯狂星期四""V我50"等关键词时，自动回复一条疯狂星期四文案——可配置概率以秦始皇口吻回复，可配置概率从 API 获取在线文案（支持图片）。

## 功能特性

- **双模式回复**：秦始皇马甲趣味话术 + API 在线文案（概率由 api_weight 控制，数据来源 [vme.im](https://vme.im/)）
- **智能关键词匹配**：支持"疯狂星期四""V我50""KFC 星期四""crazy thursday"等 20+ 种常见变体
- **图片消息支持**：自动识别 API 返回的图片链接，支持发送图文混合消息
- **API 回退机制**：API 请求失败时自动回退到秦始皇马甲话术，保证回复不中断
- **UMO 白名单**：基于统一消息来源标识控制触发范围，跨平台兼容
- **冷却防刷屏**：同一用户默认 120 秒内只回复一次，可配置
- **星期四限定**：可选仅在每周四触发
- **LLM 拦截**：触发回复后自动阻止 LLM 处理，避免重复响应

## 安装

1. 在 AstrBot WebUI 中进入 **插件管理** → **从仓库安装**
2. 搜索 `kfc_crazy_thursday` 并安装
3. 或手动克隆到 `data/plugins/` 目录后重启 AstrBot

## 配置

安装后在 WebUI 插件管理中找到本插件，点击设置进行配置：

### 白名单 (必填)

填入会话的 **UMO（统一消息来源标识）**，用英文分号 `;` 隔开。

| 场景 | UMO 格式 | 说明 |
|------|----------|------|
| 白名单某个群 | `platform:GroupMessage:群号` | 该群所有成员发言都会触发 |
| 白名单某个私聊 | `platform:FriendMessage:用户ID` | 该用户的私聊消息会触发 |

**如何获取 UMO：** 在群/私聊中发送包含关键词的消息后，查看 AstrBot 日志中 `[疯狂星期四]` 开头的调试信息，其中 `umo=` 后面的值即为 UMO。

### 冷却时间（可选）

默认 `120` 秒，同一用户在冷却时间内只回复一次。设为 `0` 则不限制。

### 仅星期四触发（可选）

默认关闭。开启后仅在每周四触发回复。

## 回复示例

> **用户**：今天是疯狂星期四，V我50！
>
> **机器人（50%）**：朕扫六合统天下，国库充盈，准了！💰
>
> **机器人（50%）**：同学你好，我第三次在图书馆看到你，请问可以加你微信嘛？我的微信号是：KFCVME50

## 工作原理

```
用户发送消息 → 正则匹配疯狂星期四关键词
  → 检查 UMO 是否在白名单中
    → 检查冷却时间
      → 50% 随机选取秦始皇马甲话术回复
      → 50% 从 vme.im API 获取在线文案回复（失败则回退）
      → 拦截 LLM 响应
```
